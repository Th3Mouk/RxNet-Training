# RabbitMQ & RxNet

How to cook the :rabbit: with 
[RxNet library](https://github.com/Domraider/rxnet) (reactive programming).

This is a simple test, to how to produce and consume messages, whatever 
interupts.

## Test if install is complete

`php run hello`

## Start producer

`php run produce`

**Options**
- orders: default `20`

## Consume messages

`php run consume`

**Options**
- type: default `simple`
